## Line editing

- `Ctrl u` тАФ Clear the current input line.
- `Ctrl k` тАФ Delete to end of line.
- `Ctrl w` тАФ Delete previous word.
- `Ctrl a` / `Ctrl e` тАФ Jump to start / end of line.
- `Alt b` / `Alt f` тАФ Move back / forward one word.
- `Ctrl y` тАФ Paste last killed text (yank).
- `Ctrl _` тАФ Undo last edit.

## History & control

- `Ctrl r` тАФ Reverse history search (repeat for older matches).
- `Ctrl g` тАФ Cancel history search.
- `Ctrl l` тАФ Clear screen.

## Power

- `sudo poweroff` тАФ Shut down now (`sudo shutdown` waits 1 min).
- `sudo reboot` тАФ Reboot now.
